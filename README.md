# COVID19 Scraping Script for Okayama

## What 's this?

岡山県公式 HP や岡山県オープンデータカタログサイトで公開されている情報を取得し、JSON として出力するスクリプトです。

[岡山県 新型コロナウイルス感染症対策サイト (非公式)](https://okayama.stopcovid19.jp) で使用する形に整形し、出力します。

## Make data

```
$ npm i
$ npm start
```
